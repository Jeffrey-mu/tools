# 日常工具网站 - 功能创意列表

这是一个工具功能清单（已完成/待开发），用于同步记录站点功能范围。

## 🛠️ 开发者工具箱 (Developer Tools)
这些工具主要面向程序员和 IT 从业者，解决开发中的高频小痛点。

1.  **JSON 格式化/校验器 (JSON Formatter)** [已完成]
    *   **功能**: 输入 JSON 字符串，进行美化（Pretty Print）、压缩、语法错误检测。
    *   **难度**: ⭐
2.  **时间戳转换 (Timestamp Converter)** [已完成]
    *   **功能**: 当前时间戳显示，Unix 时间戳与日期时间字符串（如 YYYY-MM-DD HH:mm:ss）互相转换。
    *   **难度**: ⭐
3.  **Base64 编解码 (Base64 Encoder/Decoder)** [已完成]
    *   **功能**: 文本与 Base64 互相转换，支持图片转 Base64 数据 URI。
    *   **难度**: ⭐
4.  **UUID/GUID 生成器** [已完成]
    *   **功能**: 批量生成 Version 1/4 的 UUID。
    *   **难度**: ⭐
5.  **正则表达式测试 (Regex Tester)** [已完成]
    *   **功能**: 输入正则表达式和测试文本，实时高亮匹配结果。
    *   **难度**: ⭐⭐
6.  **URL 编解码** [已完成]
    *   **功能**: 对 URL 参数进行 Encode 和 Decode。
    *   **难度**: ⭐
7.  **URL 参数解析** [已完成]
    *   **功能**: 解析 URL 查询参数，支持编辑并重组 URL。
    *   **难度**: ⭐⭐
8.  **MD5 加密** [已完成]
    *   **功能**: 计算文本 32 位 / 16 位 MD5 哈希（支持大写输出）。
    *   **难度**: ⭐
9.  **录屏 (Screen Recorder)** [已完成]
    *   **功能**: 屏幕/窗口/标签页录制，支持麦克风与系统音频（受浏览器与共享源限制）。
    *   **难度**: ⭐⭐⭐

## 📝 文本与数据处理 (Text & Data)
处理文字、格式转换等通用需求。

10. **字数/字符统计** [已完成]
    *   **功能**: 统计输入文本的汉字数、英文单词数、字符数（含/不含空格）、行数。
    *   **难度**: ⭐
11. **文本 Diff 对比** [已完成]
    *   **功能**: 左右分栏对比两段文本的差异（新增、删除、修改）。
    *   **难度**: ⭐⭐ (可使用现有库)
12. **随机密码生成器** [已完成]
    *   **功能**: 设置长度，选择包含的字符类型（大写、小写、数字、符号），生成强密码。
    *   **难度**: ⭐
13. **大小写/命名风格转换** [已完成]
    *   **功能**: camelCase, snake_case, PascalCase, kebab-case 互相转换；全大写/全小写转换。
    *   **难度**: ⭐

## 🎨 图形与设计 (Image & Design)
简单的图像处理和设计辅助工具。

14. **二维码生成器 (QR Code Generator)** [已完成]
    *   **功能**: 将文本或链接生成二维码图片，支持下载。
    *   **难度**: ⭐⭐ (使用库)
15. **颜色转换与调色板** [已完成]
    *   **功能**: HEX, RGB, HSL 互相转换；颜色选择器；生成渐变色 CSS。
    *   **难度**: ⭐⭐
16. **图片压缩 (Image Compressor)** [已完成]
    *   **功能**: 前端本地压缩图片（JPG/PNG），调整质量，减小体积。
    *   **难度**: ⭐⭐⭐

## 🏠 生活与效率 (Life & Productivity)
日常生活中可能用到的小工具。

17. **单位换算** [已完成]
    *   **功能**: 长度、重量、面积、体积、温度等常见单位换算。
    *   **难度**: ⭐
18. **番茄钟 (Pomodoro Timer)** [已完成]
    *   **功能**: 25分钟专注 + 5分钟休息的倒计时器，带简单的提示音。
    *   **难度**: ⭐⭐
19. **简单的待办清单 (Todo List)** [已完成]
    *   **功能**: 添加、删除、标记完成，数据保存在本地存储 (LocalStorage)。
    *   **难度**: ⭐⭐
20. **决策转盘/随机数生成** [已完成]
    *   **功能**: "今天吃什么"转盘，或者生成指定范围内的随机数。
    *   **难度**: ⭐⭐
